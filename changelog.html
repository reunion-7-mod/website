Version 0.9

The latest version of Reunion 7

Details

  • After a long wait, the Basic and Classic themes are finally available in the Personalization page (of Control Panel). Props to Tech Stuff for figuring it out and to Xdek for helping me implement it!
  • A better (modified) Aerexplorer has been put in place, thanks to Tech Stuff.
  • Better grammar for all readmes and instructions RTFs.
  • Removed the conventional AWM and replaced it with a Windhawk mod (thanks, Tech Stuff).
  • Moved over to DWM Tweaker (thanks, Tech Stuff).
  • Partially added support for Recovery points.
  • Implemented ALL of the publicly available Win7 CPL Shell restorations.
  • Partially restored the Windows Media Center gadget.
  • Partially restored the official Weather gadget (only for display purposes!).
  • Fixed a small inaccuracy where the "System Folder" tag was missing. Thanks, Xdek.
  • Fixed an inaccuracy where the Favorites tab was collapsed by default.
  • Fixed an inaccuracy where the Recent Places file folder was missing.
  • Fixed an inaccuracy where there was a non-functional feature to Pin something to Favorites, that button was not in W7 to begin with.
  • Fixed a small inaccuracy in the Libraries page, it used to say "Select an item to see details", now it says "... items".
  • Fixed an inaccuracy where the PowerShell ISE was that of W10, now it should be of W7.
  • Fixed a bug where the 12-hour Time Convention had a 0 next to the 1-9 hours (e.g. 09:30 → 9:30).
  • Finally, the correct, W7-accurate default programs appear in the Start Menu as intended (e.g. Solitaire, Windows Media Center).
  • Fixed a bug where the Frost accent color icon was being displayed as the Automatic color icon.
  • Fixed a bug where all power plans (excluding 'Balanced') were missing. The 'Ultimate Performance' plan has been brought back as a throwback.
  • Fixed a major issue where non-Admin accounts had broken elements of explorer (and other stuff).
  • Restored the W7 Network Projector program (for display purposes only), thanks Petya and djamaliyev64 (feedback)!
  • Restored the Windows 7 Product Key Screen, thanks ImSwordQueen and Brawllux!
  • Fixed an inaccuracy where the troubleshooting loading bar was using the Metro style instead of the Aero one, thanks Petya!
  • Fixed the inaccuracy of the OpenGlass window frames not merging properly with the rest of the glass.
  • Fixed the issue of the sad face :( on the Start Menu.
  • Disabled the "Look for an app in the Store" pop-up.
  • Ejected the leftover network drive.
  • Implemented better credits in this Readme.
  • Implemented a "Time Convention" switch.
  • Updated Winaero Tweaker.
  • Removed that buggy Windows Marketplace.
  • Implemented an extra-optimized OpenGlass [the glass rendering engine] that works 160% better. Thanks, Patrick and Tech Stuff.
  • Fixed the issue where Reunion7 was quite laggy (even after updating the drivers). The Aero Peek/Snap reflections are still present. HUGE thanks to Patrick and Tech Stuff.
  • Fixed a bug where Recent Place could crash explorer.exe.
  • Fixed a bug where Recent Places simply didn't pick up anything.
  • User Account Control (UAC) is now enabled by default (like how it's in W7).
  • Cleaned up the mess that was a collection of batch scripts to implement fixes.
  • The Weather and Currency gadgets now work. Thanks, Patrick!
  • Removed the inaccurate "Automatic" accent color.
  • Fixed extremely minute inaccuracies that the vast, vast majority of people will not notice. Thanks, Xdek.

If you want to contribute to this project, feel free to message us on our Discord server.
